## [0.1.0] - 2026-02-15

### Added
- Initial template shell with country adapter architecture
- Sweden (se) and Norway (no) sample adapters
- Country scaffolder (`npm run scaffold:country`)
- Contract test framework with golden-tests.json fixtures
- Drift detection with golden-hashes.json fixtures
- Health and version endpoint infrastructure
- CI/CD workflows (test, publish, security scanning, drift detection)
- SBOM generation (CycloneDX)
- sources.yml template for data provenance
- Issue template for data error reports
